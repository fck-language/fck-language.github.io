So what is fck?

fck is an open source coding language written in Rust

What makes it so special?

There are a few things that make fck special:

  1. You can write in any human language you want, allowing much more people an easier route into coding
  2. It has flavours! This means you only need to know fck to be able to use code for a huge variety of things. If you want to know more about flavours, then go read this page
  3. fck comes with both an interpreter and a compiler. So however you like to work, we've got you covered!
  4. You're more than likely to come across a warning before you do an error. fck doesn't like to return errors to you, so it does all it can to fix whatever's gone wrong
    This also means that circular reference errors are a thing of the past!

Where can I get started?

If you just want the basic out of box fck, then the downloads page is the best place for you. If you want to have a look at what there is to offer, then you want to go to the flavours page.
If you need to learn fck, then you'll want the tutorials page.

Can I help?

Of course! We always welcome new people to contribute to the development of fck. The best place to start is to have a look at the contribution document in the github repo, then clone the repo and see what you can do!
