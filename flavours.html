Flavours

fck has several flavours. A flavour is simply an installation of fck that has added functions, constants, and types. Each flavour is based upon the pure flavour; so anything in the pure version is in all the other flavours. All the syntax is exactly the same, so you never have to learn anything new!

Since all of this is open source, you are more than welcome to go through all the different flavours and pick and choose what you want or don't want and build a custom flavour just for you. You can also download implementations of each function, constant, or type as a file to add into your project if you need it.

Specific details about each constant, function, and type are given in the documentation. This page gives you more of an overview of the different flavours and what each one includes. Please also be careful with names of values. The names given here are not the names used in the code, but are instead names of the functions being implemented. For example the signature for the binomial coefficients function is binom_coeff(int n, int s) -> int. For the code signature, please refer to the documentation.

Counting

Counting is the maths based flavour, with an emphasis on pure maths. Statistical maths will be added in the future as a different flavour.
